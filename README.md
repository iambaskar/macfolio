# macfolio

A fully interactive **macOS-inspired portfolio** built with React.js. Turn your browser into a playful desktop environment where you can drag windows, play music, write notes, browse photos, and explore my work.

## Features

- **Realistic macOS Interface**
  - Draggable windows with traffic light controls (hover to reveal × − +)
  - Glassmorphism + backdrop blur effects
  - Dynamic menu bar with live clock and date
  - Magnifying dock with active app indicators

- **Interactive Windows & Widgets**
  - **Finder** – Recents and Applications browser
  - **Music Player** – Play real tracks via YouTube (Radiohead, Nirvana, Cigarettes After Sex, etc.)
  - **Notes** – Create, edit, and delete sticky notes
  - **Mail** – Contact form with simulated sending animation
  - **Photos** – Personal photo gallery with lightbox viewer
  - **About Me** – Detailed profile with tabs (Profile, Experience, Skills, Education, Projects)
  - **Awwwards Honors** – Award showcase
  - **Desktop Cards** – Flippable project cards with 3D effect

- **Smooth Animations**
  - Window pop-in animations
  - 3D card flip
  - Dock icon magnification
  - Drifting clouds in background

## Tech Stack

- **Frontend**: React.js (Hooks)
- **Styling**: Pure CSS (Glassmorphism + custom animations)
- **Animations**: CSS keyframes + React state
- **Music**: YouTube Embed API
- **No external UI libraries** – Everything built from scratch
